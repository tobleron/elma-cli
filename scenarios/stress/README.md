This suite stress-tests Elma's runtime behavior beyond simple routing.

It is designed to tune and calibrate:
- speech act discrimination
- workflow and mode routing
- scope narrowing
- formula selection
- evidence compaction
- artifact classification
- claim checking
- presentation quality
- structural handling of write and execute requests

The files here are intentionally grounded in the real `elma-cli` workspace so tuning can optimize the active model against the current local environment.
