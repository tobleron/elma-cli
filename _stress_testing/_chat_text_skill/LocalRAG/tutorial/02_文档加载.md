# 第二章：文档加载（Loader）

## 目标

把 PDF、TXT、Markdown 文件读入 Python，变成干净的文本字符串，为后续分块做准备。

## 难点在哪里？

不同格式的文档结构差异很大：

- **TXT**：纯文本，最简单
- **Markdown**：有标题、列表、代码块结构
- **PDF**：最复杂 — 有扫描件（图片）、表格、多栏布局、页眉页脚

我们先处理最简单的情况：**文本文件和标准 PDF**。

## 代码实现

```python
# src/loader.py

from pathlib import Path
from typing import Optional, Literal

# ============================================================
# 基础：按文件扩展名分发
# ============================================================

def load_document(file_path: str) -> str:
    """根据文件类型调用不同的加载器。"""
    path = Path(file_path)
    suffix = path.suffix.lower()

    if suffix == ".pdf":
        return _load_pdf(file_path)
    elif suffix in (".txt", ".md"):
        return _load_txt(file_path)
    else:
        raise ValueError(f"不支持的文件格式: {suffix}")


# ============================================================
# 核心：文本文件加载（最简单的部分）
# ============================================================

def _load_txt(file_path: str) -> str:
    """
    读取纯文本或 Markdown 文件。
    使用 UTF-8 编码，失败则尝试 GBK（部分中文文档用 GBK）。
    """
    encodings = ["utf-8", "gbk", "gb2312", "latin1"]

    for enc in encodings:
        try:
            with open(file_path, "r", encoding=enc) as f:
                return f.read()
        except UnicodeDecodeError:
            continue

    raise ValueError(f"无法解码文件: {file_path}")


# ============================================================
# 核心：PDF 加载（相对复杂）
# ============================================================

def _load_pdf(file_path: str) -> str:
    """
    使用 pypdf 库读取 PDF 文本内容。
    注意：pypdf 只读取文字型 PDF，扫描件（图片）无法读取。
    """
    from pypdf import PdfReader

    reader = PdfReader(file_path)
    texts = []

    for i, page in enumerate(reader.pages):
        text = page.extract_text()
        if text and text.strip():
            # 去除页眉页脚（常见模式：连续空格+页码）
            text = _clean_page_text(text)
            texts.append(text)

    # 用双换行符连接各页，保留页面边界信息
    return "\n\n".join(texts)


def _clean_page_text(text: str) -> str:
    """
    清理 PDF 提取的原始文本中常见的噪音。
    """
    import re

    # 1. 去除连续空格
    text = re.sub(r" {2,}", " ", text)

    # 2. 去除行首行尾多余空白
    lines = [line.strip() for line in text.split("\n")]
    text = "\n".join(lines)

    # 3. 去除孤立页码（单独一行的数字）
    text = re.sub(r"(?<=\n)\s*\d+\s*(?=\n)", "\n", text)

    return text.strip()
```

## 关键设计点讲解

### 1. 按扩展名分发（Factory 模式）

```python
def load_document(file_path: str) -> str:
    suffix = Path(file_path).suffix.lower()
    if suffix == ".pdf":      return _load_pdf(file_path)
    elif suffix in (".txt", ".md"): return _load_txt(file_path)
```

这样新增文件格式只需要加一个 `elif`，不影响现有代码。

### 2. PDF 文本清理

PDF 提取的文本有很多"噪音"：

| 噪音类型 | 示例 | 处理方式 |
|---------|------|---------|
| 连续空格 | `"这    是    文"` | 正则替换为单空格 |
| 孤立页码 | `"\n  42  \n"` | 正则删除单独一行的数字 |
| 换行碎句 | 一句话被从中间断开 | 保留原换行，后续 chunker 处理 |

### 3. 编码兜底

中文文档编码不统一，有的用 UTF-8，有的用 GBK：

```python
for enc in ["utf-8", "gbk", "gb2312", "latin1"]:
    try:
        return open(file_path, encoding=enc).read()
    except UnicodeDecodeError:
        continue
```

## 局限性

这个 Loader 有以下已知局限，理解它们很重要：

1. **扫描 PDF（图片型）无法读取** — 需要 OCR（如 pytesseract），本项目不处理
2. **表格无法正确结构化** — PDF 表格被读成混乱的文本
3. **特殊格式（Word/Excel）不支持** — 可以转成 PDF 或 TXT 再处理

## 扩展：支持更多格式

如果未来需要支持 Word 文档（.docx），安装 `python-docx` 后加一行即可：

```python
elif suffix == ".docx":
    from docx import Document
    doc = Document(file_path)
    return "\n".join([p.text for p in doc.paragraphs])
```
